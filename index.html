Bienvenido a la tienda online

Este es un proyecto que surgio como una idea, para hacer un plataforma de tienda online y venta de articulos por medio de internet este sitio web esta echo en: HTML, JAVASCRIPT,CSS, MYSQL y PHP.

Aqui vas a encontrar varias sesiones, una de inicio de bienvenida, otra de compra de ropa, de compra de objetos tecnologicos, una de promociones, una de contacto y la de cerrar sesion. El formulario de contacto va ser un medio de comunicacion entre el usuario que entre al sitio y el respectivo autor o creador de la plataforma, esto con el fin de generar criticas constructivas para promover la mejora de acuerdo al diseño, modelado, creatividad, recomendaciones, etc.

Esta por el momento es una version beta del programa, y cuando se haya finalizado se subira al hosting, con su respectiva base de datos para un manejo mas funcional. Todo lo encontrado de acuerdo a contenido es solo pura especulacion, para fines de prueba, asi que no tomar en cuenta.
